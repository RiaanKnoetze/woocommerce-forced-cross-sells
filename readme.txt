Force specific WooCommerce products to always show as cross-sells on the cart page, regardless of individual product cross-sell configurations.

== Description ==

WooCommerce Forced Cross-Sells allows you to display a specific set of products as cross-sells on the cart page, overriding any individual product cross-sell settings. This is particularly useful for promoting certain products that complement the customer's cart contents.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/woocommerce-forced-cross-sells` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the WooCommerce->Settings->Products->Cross-sells screen to configure the plugin.

== Frequently Asked Questions ==

= Can I choose multiple products to display as cross-sells? =

Yes, you can select multiple products that you want to always appear as cross-sells in the cart.

= Will this plugin override individual product cross-sell settings? =

Yes, the cross-sell products you select in the plugin settings will override any cross-sell configurations set on individual products.
